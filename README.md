# Dashboard UFPR

Dashboard interativo para o Hackathon de dados da UFPR

Este dashboard permite visualizar e analisar dados das pesquisas
realizadas pela UFPR:  
- Cursos
- Disciplinas presenciais  
- Disciplinas EAD
- Pesquisa institucional

## Como Executar

### Pré-requisitos
- Python 3.7 ou superior
- pip (gerenciador de pacotes do Python)

### Instalação e Execução

1. **Clone o repositório**
   ```bash
   git clone Dashboard_Hackathon
   cd Dashboard
   pip install -r requirements.txt
   cd src
   python app.py (python3 app.py no Linux/MacOS)
