# Dashboard UFPR

Dashboard interativo para o Hackathon de dados da UFPR

Este dashboard permite visualizar e analisar dados das pesquisas
realizadas pela UFPR:  
- Cursos
- Disciplinas presenciais  
- Disciplinas EAD
- Pesquisa institucional

## Como Executar

### Pré-requisitos
- Python 3.7 ou superior
- pip (gerenciador de pacotes do Python)

### Instalação e Execução

1. **Clone o repositório**
   ```bash
   git clone https://github.com/benaytms/dashboard.git
   cd dashboard/
   python3 -m venv venv
   pip install -r requirements.txt
   cd src/
   python3 app.py

2. **Abrir no navegador**: http://localhost:8050
